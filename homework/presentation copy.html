Advanced Typography

Typography can be complex and very detailed. For this Presentation we are going to focus on the basics of typography for websites. We are not going to worry about actually creating fonts.

Basic Typography

There are four different classifications of type. They are determine by their style of the individual letters. We are going to learn how they differ from each other and how they work together.

Serif Font

This is the classic "Times New Roman" the comes out with a little flair at the ends, but very minimal. It has an older look to it rather than modern. This is a great font for either headers or body content.

Sans Serif Font

Sans serif is just like serif except with out the flair completly. It has blunt ends and has a very modern look. This is a great font for both headers or body content. I personally like this one the most because it has a clean look.

Script Font

This font looks very similar to fancy handwriting. It can give a interesting personal look with these fonts. These fonts are really only good for headers because it can be difficult to read if it too small and use in body content.

Serif Font

Decorative is just like script. Every letter has it's own decoration that can be subtle or wild. It can add personality to your website. This is another font style to only use for headers not body content.

Creating Contrast with Fonts

You can create contrast with type in differents ways.

  • Different size font next to each other. Size of header compare to body font.
  • Putting different font styles together. For example, Sans Serif agains Serif creates great contrast.
  • Adding bold or italics can create contrast in your font.

Do's and Don'ts in Typography

    Do's

  • Explore new fonts on websites like...
  • fonts.google.com
  • typekit.com
  • https://www.behance.net/collection/4860923/Free-Fonts
  • Have more than one font on your website, but no more than three

    Don'ts

  • Don't use common fonts like...
  • Comic Sans MS
  • Papyrus
  • Bradley Hand
  • Don't put two fonts together that have the same type classification
  • Don't use decorative and script fonts for body copy
